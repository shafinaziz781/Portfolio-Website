✨ Features
•	Loading Screen — Pulsing SA initials with 3 expanding ripple rings + animated progress bar that fades out on page load. <br>
•	Custom Glowing Cursor — Cyan dot with a drilling ring that pulses outward, transforms on hover. <br>
•	Live Network Canvas — Interactive particle network in the background, nodes repel from mouse in real time (Canvas API). <br>
•	Hero Section — Shimmer gradient text animation, 3D floating cyber car with continuous rotation. <br>
•	Scroll Reveal Animations — Fade-up, slide-left, slide-right, scale-in triggered by IntersectionObserver. <br>
•	Animated Skill Bars — Animate from 0% only when scrolled into view. <br>
•	Infinite Companies Ticker — Smooth infinite scroll, pauses on hover.<br>
•	Project Cards — Top-edge gradient borders, mouse-tracking radial glow, ripple on hover. <br>
•	Pulsing Ripple Rings — On the contact section, expanding rings animate continuously. <br>
•	Fully Responsive — Single column on mobile, hamburger nav, cursor hidden on touch devices. <br>
